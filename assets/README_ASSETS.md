# Assets Required

This directory should contain the following files:

## Required Images

1. **cover_passport.jpg** (Recommended: 600x800px or 3:4 aspect ratio)
   - Cover image for the passport-style first page
   - Should be optimized for web (JPEG quality 80-85%)

2. **map_image.jpg** (Recommended: 700x400px or similar)
   - Map showing the venue location
   - Can be a screenshot from Google Maps or custom map

3. **qr_map.png** (Recommended: 150x150px minimum)
   - QR code that links to Google Maps directions
   - Generate using: https://www.qr-code-generator.com/ or similar

4. **logo.png** (Optional, Recommended: 96x96px)
   - Publisher logo for AMP Story metadata
   - Square format preferred

## Fonts (Optional - will fallback to system fonts if not provided)

The following fonts are referenced but will gracefully fallback if not present:

1. **NotoSerifTelugu-Regular.woff2** - For Telugu text
2. **NotoSerifTelugu-Bold.woff2** - For Telugu bold text
3. **PlayfairDisplay-Regular.woff2** - For English text
4. **PlayfairDisplay-Bold.woff2** - For English bold text

### Font Sources:
- Noto Serif Telugu: https://fonts.google.com/noto/specimen/Noto+Serif+Telugu
- Playfair Display: https://fonts.google.com/specimen/Playfair+Display

To convert TTF/OTF to WOFF2, use: https://cloudconvert.com/ttf-to-woff2
